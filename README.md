# 📊 Expense Tracker

A modern and responsive **Expense Tracker Web Application** that helps users manage their daily and monthly expenses efficiently.

Instead of writing all expenses in a diary book, users can simply add their expenses into this application and easily analyze how much they have spent throughout the month. This saves a lot of time, reduces confusion, and provides better financial clarity.

---

## 🚀 Features

- ➕ Add new expenses  
- 📋 View list of all recorded expenses   
- 💰 Automatically calculate total expenses  
- 📊 Interactive dashboard with graphical representation  
- 📅 Monthly expense analysis

--- 

## 📈 Dashboard

This application includes a **Dashboard** that visually represents expenses in **graphical format**.

- Expenses are displayed using charts
- Users can easily analyze monthly spending
- Visual representation makes tracking simple and clear
- Helps user to identify spending patterns

---
## 💻 Tech Stack
- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Dashboard:** Chart.js
- **Language:** JavaScript (JSX)
- **Version Control:** Git & Github

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/madupojumounika/expense-tracker.git
cd expense-tracker
```
### 2️⃣ Install Dependencies
```
npm install
```
### 3️⃣ Run the project
```
npm run dev
```
- Open your browser and go to
  ```
  http://localhost:5173
  ```
